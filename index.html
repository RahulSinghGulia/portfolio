Research Highlights

             
         

                          1. White-Box Modeling of V2X Link Performance Using Stabilized Symbolic Regression                      

         

            This work introduces the Stabilized Symbolic Regression Framework for V2X (**SSRV**) to derive a **compact, analytically interpretable** expression for Block Error Rate (BLER) prediction in high-mobility V2X channels (5G Sidelink).          

         
               
  • **The Challenge:** Traditional Deep Neural Networks (DNNs) achieve high accuracy but are **black boxes** and computationally expensive, limiting real-time deployment in safety-critical V2X systems.
    •            
  • **The Solution (SSRV):** Uses Genetic Programming and a physics-informed $\mathbf{Y = -\ln(\mathrm{BLER})}$ transformation to discover a closed-form formula (158 nodes) that captures complex, non-linear dependencies on **SNR, relative velocity, DMRS symbols**, and **MCS**.
    •            
  • **Key Performance:** Achieves high predictive performance with **$R^2 = 0.8684$** and **Mean Squared Error $= 2.08 \times 10^{-2}$** on the test set, offering **comparable accuracy to neural networks** but with full interpretability and ultra-fast inference suitable for embedded deployment.
    •            
  • **Value:** Provides a powerful tool for **real-time adaptive resource allocation** and rapid scenario evaluation in dynamic V2X environments. Sensitivity analysis confirmed SNR and relative velocity as dominant factors.
    •

                          2. A Memory-Augmented Generative Framework for Spatio-Temporal mmWave Channel Prediction                      

         

            Evo-WISVA is a hybrid Generative AI model addressing the core challenge of real-time spatio-temporal mmWave channel prediction in dynamic industrial environments (Industry 4.0). Autonomous operations require deterministic, real-time quality-of-service, but the high-frequency wireless channel is inherently non-deterministic.          

         
               
  • **Innovation:** Utilizes a memory-augmented Generative AI architecture to predict SINR and interference patterns, creating self-learning Wireless Digital Twins.
    •            
  • **Quantified Performance:** Achieves **1.75 dB Mean Absolute Error (MAE)** accuracy, significantly outperforming traditional methods (Ray-Tracing/NS-3) which are computationally intractable.
    •            
  • **Efficiency:** Delivers superior prediction performance with a **sub-millisecond per-frame prediction time**, crucial for **URLLC** (Ultra-Reliable Low-Latency Communication) and proactive network adaptation.
    •            
  • **Robustness:** The model is highly robust to limited training data and effectively adapts to rapid signal quality changes due to dynamic Line-of-Sight (LoS) and mobile obstacles.
    •          
       
             
         

                          3. CSMA/CA Contention Window Simulation (MAC Layer)                      

         

            This project presents a Python-based discrete-event simulation of the CSMA/CA (Carrier Sense Multiple Access with Collision Avoidance) protocol, a fundamental MAC layer method used in wireless networks like Wi-Fi. Built using simpy, it models and analyzes network behavior under various configurations to study the dynamics of RTS/CTS handshakes, collision avoidance, and exponential backoff algorithms.           

           
                 
  • Simulates carrier sensing, DIFS/SIFS, ACKs, and collision handling with retransmission logic.
    •              
  • Evaluates the impact of RTS/CTS mechanisms and node density on throughput and latency.
    •              
  • Includes adjustable parameters such as coverage area, backoff limits, and timing windows.
    •              
  • Generates plots for Throughput vs. Nodes, Collisions vs. Nodes, and Efficiency vs. Backoff Limit.
    •            
         
                      
         

                          4. UAV Low Altitude Air-to-Ground (U-LAAG) Path Loss Model                      

         

            My M.S. thesis research introduced a non-deterministic statistical path loss model for indoor wireless communication in UAV, IoT, and cloud robotics applications.              The proposed **U-LAAG model**, adapted from the ITU-R framework, accounts for low-altitude UAV operations within the 2.4–2.5 GHz ISM band.           

           
                 
  • Addresses path loss variations due to materials, floor plans, and multipath scattering in indoor environments.
    •              
  • Validated through extensive experiments in university buildings, residential dwellings, and basements.
    •              
  • Compared against ITU-R, Two-Ray, and Log-Distance models — U-LAAG closely matched empirical drive test results.
    •              
  • Provides a basis for accurate indoor Wi-Fi and UAV network design for real-world deployments.
    •

Research Highlights

2. White-Box Modeling of V2X Link Performance Using Stabilized Symbolic Regression

Stabilized symbolic regression framework for interpretable BLER modeling in 5G V2X sidelink communications, achieving neural-level accuracy with fully analytical, deployment-ready expressions.

3. UAV Low-Altitude Air-to-Ground (U-LAAG) Path Loss Model

My M.S. thesis research introduced a non-deterministic statistical path loss model for indoor wireless communication in UAV, IoT, and cloud robotics applications. The proposed U-LAAG model, adapted from the ITU-R framework, accounts for low-altitude UAV operations within the 2.4–2.5 GHz ISM band.

  • Addresses path loss variations due to building materials, floor plans, and multipath scattering.
  • Validated through extensive experiments in university buildings, residential dwellings, and basements.
  • Compared against ITU-R, Two-Ray, and Log-Distance models — U-LAAG closely matched empirical drive-test results.
  • Provides a practical basis for accurate indoor Wi-Fi and UAV network design.

Research Projects

1. A Memory-Augmented Generative Framework for Spatio-Temporal mmWave Channel Prediction in Dynamic Industrial Environments

Predicting real-time wireless network performance is computationally expensive and fundamentally limited by the static nature of traditional tools. Physical measurements, ray tracing, and NS-3 simulations require hours per snapshot and fail to capture rapid spatio-temporal variations caused by mobile robots, blockage, and dynamic LOS/NLOS transitions.

To address this gap, I developed Evo-WISVA, a memory-augmented generative AI framework for real-time spatio-temporal mmWave channel prediction. The model fuses learned latent representations with historical memory to enable proactive, near-real-time prediction of wireless performance in cluttered industrial environments.

  • High Accuracy: Achieves 1.75 dB MAE, outperforming state-of-the-art benchmarks.
  • Robustness: Generalizes effectively to unseen layouts, robot trajectories, and channel dynamics.
  • Efficiency: Sub-millisecond per-frame prediction latency, enabling real-time deployment.

This work enables self-optimizing wireless networks and provides the prediction fidelity required for URLLC, Industry 4.0, and autonomous factory operations.

R. Gulia et al., “Memory-Augmented Generative AI for Real-Time Wireless Prediction in Dynamic Industrial Environments,” IEEE Transactions on Industrial Informatics (Under Review).

R. Gulia et al., “AI-Driven Radio Propagation Prediction in Automated Warehouses using Variational Autoencoders,” IEEE Transactions on Industrial Informatics (Under Review).

2. White-Box Modeling of V2X Link Performance Using Stabilized Symbolic Regression

Autonomous V2X systems require deterministic, interpretable, and ultra-fast link performance prediction. While deep neural networks achieve high accuracy, they operate as black boxes and impose non-trivial inference costs, limiting their suitability for safety-critical and embedded V2X applications.

This work introduces a Stabilized Symbolic Regression framework that discovers compact, closed-form analytical expressions for BLER prediction directly from realistic 5G V2X link-level simulation data.

  • 158-node analytical expression enabling ultra-fast inference.
  • Test-set performance: R² = 0.8684, MSE = 2.08×10⁻².
  • Numerically protected operators ensure stability across SNR and mobility ranges.
  • Residual and sensitivity analysis confirms physical consistency.

The resulting model offers neural-level accuracy with full interpretability, making it suitable for real-time adaptive resource allocation and V2X system design.

R. Gulia et al., “White-Box Modeling of V2X Link Performance Using Stabilized Symbolic Regression,” IEEE Transactions on Vehicular Technology (Under Review).

3. UAV Low-Altitude Air-to-Ground (U-LAAG) Path Loss Model

My M.S. thesis introduced a non-deterministic statistical path loss model for indoor UAV, IoT, and cloud robotics communication. The proposed U-LAAG model, adapted from the ITU-R framework, explicitly addresses low-altitude UAV operations in the 2.4–2.5 GHz ISM band.

  • Captures path loss variations due to materials, floor plans, and multipath scattering.
  • Validated via extensive measurements in university buildings, residential homes, and basements.
  • Outperforms ITU-R, Two-Ray, and Log-Distance models in matching empirical data.
  • Provides a foundation for realistic indoor Wi-Fi and UAV network planning.

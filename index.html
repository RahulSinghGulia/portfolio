Research Projects

1. A Memory-Augmented Generative Framework for Spatio-Temporal mmWave Channel Prediction in Dynamic Industrial Environments

Predicting real-time wireless network performance is time consuming and resource intensive. Traditional tools such as ray tracing, NS-3, and physical measurements require hours per snapshot and cannot support proactive adaptation in dynamic industrial environments.

I developed Evo-WISVA, a hybrid memory-augmented generative AI framework for real-time spatio-temporal mmWave channel prediction, enabling self-optimizing wireless networks.

  • Accuracy: 1.75 dB MAE, outperforming state-of-the-art benchmarks.
  • Robustness: Adapts to unseen layouts, mobility patterns, and channel conditions.
  • Efficiency: Sub-millisecond per-frame prediction time.

R. Gulia et al., “Memory-Augmented Generative AI for Real-Time Wireless Prediction in Dynamic Industrial Environments,” IEEE Transactions on Industrial Informatics (Under Review).

R. Gulia et al., “AI-Driven Radio Propagation Prediction in Automated Warehouses using Variational Autoencoders,” IEEE Transactions on Industrial Informatics (Under Review).

2. White-Box Modeling of V2X Link Performance Using Stabilized Symbolic Regression

This work proposes a stabilized symbolic regression framework to derive compact, interpretable analytical expressions for BLER prediction in 5G V2X systems.

  • 158-node closed-form analytical model.
  • Test performance: R² = 0.8684, MSE = 2.08 × 10⁻².
  • Ultra-fast inference suitable for embedded deployment.

R. Gulia et al., “White-Box Modeling of V2X Link Performance Using Stabilized Symbolic Regression,” IEEE Transactions on Vehicular Technology (Under Review).

3. UAV Low Altitude Air-to-Ground (U-LAAG) Path Loss Model

My M.S. thesis introduced a non-deterministic statistical path loss model for indoor UAV, IoT, and cloud robotics communication in the 2.4–2.5 GHz ISM band.

  • Accounts for materials, floor plans, and multipath scattering.
  • Validated through extensive indoor measurement campaigns.
  • Outperforms ITU-R, Two-Ray, and Log-Distance models.
  • Enables accurate indoor UAV and Wi-Fi network design.

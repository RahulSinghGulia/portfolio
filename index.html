About Me

My research lies at the intersection of wireless communication, machine learning, and system-level modeling, with a focus on enabling deterministic, low-latency, and interpretable wireless intelligence for autonomous vehicles, industrial automation, and next-generation networks.

Research Highlights

1. A Memory-Augmented Generative Framework for Spatio-Temporal mmWave Channel Prediction in Dynamic Industrial Environments

This Ph.D. research introduces Evo-WISVA, a memory-augmented generative AI framework for real-time spatio-temporal mmWave channel prediction. The model enables self-evolving wireless digital twins capable of proactive network optimization.

  • High Accuracy: 1.75 dB MAE, outperforming state-of-the-art models.
  • Robustness: Generalizes to unseen environments and limited data.
  • Efficiency: Sub-millisecond per-frame inference.

Related Papers:
R. Gulia et al., “Memory-Augmented Generative AI for Real-time Wireless Prediction in Dynamic Industrial Environments,” IEEE Transactions on Industrial Informatics (Under Review).
R. Gulia et al., “AI-Driven Radio Propagation Prediction in Automated Warehouses using Variational Autoencoders,” IEEE Transactions on Industrial Informatics (Under Review).

2. White-Box Modeling of V2X Link Performance Using Stabilized Symbolic Regression

This work proposes a stabilized symbolic regression framework for white-box modeling of block error rate (BLER) in 5G V2X sidelink communications. The approach derives compact, interpretable analytical expressions that capture complex non-linear dependencies on SNR, mobility, and PHY-layer parameters.

  • Interpretability: Fully analytical BLER model with physical insight.
  • Efficiency: 158-node expression enabling ultra-fast inference.
  • Accuracy: Achieves R² = 0.8684, comparable to neural networks.

The framework bridges the gap between high-fidelity link-level simulations and real-time system-level evaluation, enabling adaptive resource allocation and embedded V2X deployment.

Paper:
R. Gulia et al., “White-Box Modeling of V2X Link Performance Using Stabilized Symbolic Regression,” IEEE Transactions on Vehicular Technology (Under Review).

3. CSMA/CA Contention Window Simulation (MAC Layer)

Discrete-event simulation of IEEE 802.11 CSMA/CA using SimPy, enabling performance analysis of contention behavior, RTS/CTS, and exponential backoff mechanisms.

4. UAV Low-Altitude Air-to-Ground (U-LAAG) Path Loss Model

A statistical indoor path loss model for low-altitude UAV and IoT communications, validated through extensive real-world measurements and shown to outperform traditional ITU-R-based models.
